# Rohith Sagar Karnala portfolio

Static portfolio site for GitHub Pages.

## Local preview

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Deploy on GitHub Pages

1. Use the repository `rohitsagar363.github.io`.
2. Push this repo to the `main` branch.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Build and deployment` to `Deploy from a branch`.
5. Select `main` and `/ (root)`.

The site will publish at `https://rohitsagar363.github.io`.
